选中变量名快速提供创建类型，比如ref、computed、methods、 function、 watch、 reactive,等等，根据当前是vue2还是vue3，生成对应的变量，并跳转到新创建的变量位置

![demo](/assets/demo.gif)

## 按键
- win: ctrl+g
- mac: cmd+g
